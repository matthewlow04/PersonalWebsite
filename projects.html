Projects

Fastbreak

iOS app made using SwiftUI aimed at enhancing pick up basketball. Dynamically scorekeepers by tracking team and individual player statistics through an easy to navigate and intuitive UI. Player data is saved locally using CoreData with player achievements and the ability to easily add, edit, and delete players. MVVM architecture is used to improve code readability, scalability and testability with XCTest unit tests.

Technologies: Swift, SwiftUI, CoreData, XCTest

AutoSpotter

iOS car spotting app made using SwiftUI which lets users identify cars from the street using a classification CoreML model which can differentiate between 431 unique car models. Users can then save and track their cars and add notes and additional information to their car. Users can login with Firebase to view their cloud car data which is stored using Firestore.

Technologies: Swift, SwiftUI, CoreML, Firebase, Firestore

Weather App

iOS weather app made using SwiftUI built as an improvement over Apple's weather app. Made to be more vibrant, customizable and readable with dark and light mode support. Gathered weather data using the Weatherbit API and Alamofire to handle link requests. Fetched JSON information is then parsed into live detailed weather information.

Technologies: Swift, SwiftUI, Cocoapods, Alamofire
